## 3. USB 协议规范集合定位

- 落地位置：`06_Reference/USB/`（定位为纯参考资料，不涉及实现代码）。
- 范围：核心协议（USB 2.0/3.x 设备框架）+ 常见 Class 协议（HID / Mass Storage / Audio-Video / CDC）
  + 抓包分析方法论，覆盖"识别抓包数据所需的协议层次"这一目标。
- 版权处理：不转载 USB-IF/各 Class 规范原文，仅整理公开可查的结构性知识点（传输类型、描述符字段、
  常见请求/命令），并附官方规范链接供查阅。
- 与 `PS-0001` 的关系：`Relates-To: PS-0001`，作为该 Problem Statement 所列"外设交互通信"痛点领域的
  知识准备工作；未来若该领域内容积累到一定规模，将按 ADR-0001 流程在 `04_Runtime/Capability/USB/`
  建立正式的 ADR/RFC/Prototype/SDK 结构。
